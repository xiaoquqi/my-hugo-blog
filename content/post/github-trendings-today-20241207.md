TiKV 是一个开源的、分布式的、支持事务的键值数据库，旨在补充 TiDB。它不仅提供了经典的键值 API，还提供了符合 ACID 的事务 API。TiKV 使用 Rust 构建，并由 Raft 驱动，受到了谷歌分布式系统和学术成就的启发。它保证了数据一致性，支持地理复制、水平扩展、一致的分布式事务、协处理器支持，并与 TiDB 协同工作。
