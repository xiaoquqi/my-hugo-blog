Omost 是一个旨在将LLM的编码能力转换为图像生成能力的项目。它提供了预训练的LLM模型，这些模型编写代码来组合图像视觉内容，使用Omost的虚拟Canvas代理进行渲染，然后可以通过特定的图像生成器实现生成图像。
