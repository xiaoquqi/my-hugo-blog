Prometheus是一个开源的监控系统和时间序列数据库。它从配置的目标收集指标，评估规则表达式，显示结果，并在观察到指定条件时触发警报。其特点包括多维数据模型、强大的查询语言（PromQL）、不依赖于分布式存储、支持拉取和推送模型的时间序列收集、通过服务发现或静态配置进行目标发现、多种图形和仪表板支持，以及对层次和水平联邦的支持。
