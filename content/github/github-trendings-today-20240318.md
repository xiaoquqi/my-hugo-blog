Chronos是基于语言模型架构的预训练时间序列预测模型系列。它通过缩放和量化将时间序列转换为一系列标记，并使用交叉熵损失在这些标记上训练语言模型。一旦训练完成，通过在历史上下文中对多个未来轨迹进行采样，可以获得概率预测。Chronos模型已经在大量公开可用的时间序列数据以及使用高斯过程生成的合成数据上进行了训练。
