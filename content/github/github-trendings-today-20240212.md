Hurl是一个多功能的命令行工具，用于使用纯文本运行和测试HTTP请求。它可用于单个和基于会话的请求，并支持链式多个请求、捕获值以及对标头和主体响应进行查询评估。由libcurl支持，Hurl快速、高效，并且支持HTTP/3。它还提供各种测试功能，如不同类型的查询和断言、测试HTTP端点、检查响应字节，并生成各种格式的报告。
