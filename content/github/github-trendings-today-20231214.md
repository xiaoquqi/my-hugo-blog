PageSpy 是一款用于调试远程 Web 项目的工具。它基于对原生 API 的封装，在调用原生方法时对参数进行过滤和转换，将其整理成格式规范的消息，以便调试端消费。调试端接收消息数据，并提供类似控制台的交互界面来呈现数据。
