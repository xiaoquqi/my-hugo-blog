MaxKB 是基于 LLM 大语言模型的知识库问答系统。它旨在成为一种开箱即用的解决方案，支持轻松集成到第三方业务系统中。它允许直接上传文档、自动网络文档爬取、文本分割、向量化，并提供智能问答交互体验。它可以零编码快速嵌入到第三方业务系统，并支持多个大型模型，包括本地私有模型如 Llama 2、Azure OpenAI 和百度千帆大模型。
