一种可滚动的平铺式Wayland合成器，将窗口排列在无限向右滚动的列中，打开新窗口不会导致现有窗口调整大小。每个显示器都有自己独立的窗口条，窗口永远不会“溢出”到相邻的显示器。工作区是动态的，垂直排列，每个显示器都有独立的工作区集，并始终存在一个空的工作区。工作区布局在断开和连接显示器时会得到保留。
