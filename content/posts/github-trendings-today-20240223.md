![](/images/speykious-cve-rs-0.png)

cve-rs 是一个使用 100% 安全的 Rust 编写的开源项目，允许以内存安全的方式引入常见的内存漏洞到 Rust 程序中。它旨在提供一种在不损害程序安全性的情况下破坏程序内存的方法。该项目致力于通过在整个代码库中使用 #![deny(unsafe_code)] 并实现内存漏洞和某些 Rust 函数的安全重新实现来确保内存安全。
